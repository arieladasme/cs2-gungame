# Manual: crear cuenta Ko-fi (donaciones cs2-watch)

> Objetivo: activar `ko-fi.com/arieladasme` — los links ya apuntan ahí desde el README, el `.github/FUNDING.yml` y el ☕ del header del panel. **Si eliges otro handle, avisar a Claude pa' ajustar los 3 lugares.**

## Por qué Ko-fi
- **0% comisión** en donaciones únicas (solo cobra la pasarela de pago).
- Estándar en la comunidad gaming/modding ("buy me a coffee" sin el 5% de BMAC).
- Sin aprobación previa — queda activo el mismo día.

## Pasos (~10 min)

1. **Crear cuenta**: [ko-fi.com](https://ko-fi.com) → *Sign up* (email o Google).
2. **Handle**: en *Settings → Edit Profile* fijar la URL de la página como **`arieladasme`** → queda `ko-fi.com/arieladasme`.
3. **Perfil**: foto/avatar, nombre y descripción corta. Sugerencia (inglés, es proyecto OSS):
   > *Building cs2-watch — an open-source HLSW-style live admin panel for CS2 servers. Coffee keeps the commits coming ☕*
4. **Cobros**: *Settings → Payment options* → conectar **PayPal** (lo más simple desde Chile) o Stripe si está disponible pa' tu país. Verificar la lista oficial de países en la página de Ko-fi — no asumir.
5. **Donación**: configurar el monto del "café" (default US$3–5 está bien) y el mensaje de agradecimiento automático.
6. **Verificar los 3 puntos de entrada**:
   - Badge Ko-fi del README → `github.com/arieladasme/cs2-watch`
   - Botón *Sponsor* del repo (aparece porque `FUNDING.yml` ya lista `ko_fi: arieladasme`)
   - ☕ del header del panel cs2-watch
7. **Opcional**: Ko-fi Gold (suscripción de pago) da features extra (metas, membresías); NO es necesario — las donaciones base ya son 0%.
