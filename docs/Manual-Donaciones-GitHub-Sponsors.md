# Manual: enrolarse en GitHub Sponsors (donaciones cs2-watch)

> Objetivo: activar `github.com/sponsors/arieladasme` — el `.github/FUNDING.yml` del repo ya lo lista; al aprobarte, el botón **Sponsor** aparece solo en `cs2-watch` (y en todos tus repos).

## Por qué GitHub Sponsors
- **0% comisión** — GitHub no cobra nada, ni pasarela (paga GitHub).
- Es el canal "serio" de OSS: aparece integrado en el repo, perfil y releases.
- Complementa a Ko-fi: Sponsors pa' devs/empresas, Ko-fi pa' gamers casuales.

## Pasos (~20 min + días de aprobación)

1. **Solicitar**: [github.com/sponsors](https://github.com/sponsors) → *Get sponsored* con la cuenta `arieladasme`.
2. **País**: el pago va vía **Stripe Connect**. Verificar que Chile esté en la lista oficial de regiones soportadas (la lista crece con el tiempo — revisar en la propia página de signup, no asumir). Si no estuviera: Ko-fi queda como canal único mientras tanto.
3. **Onboarding de Stripe**: identidad (RUT/pasaporte), datos bancarios pa' los payouts. Es el paso más largo.
4. **Perfil de sponsor**:
   - Bio corta: qué haces (cs2-watch, plugins CS2, OSS gaming).
   - **Tiers sugeridos** (simple, no inflar):
     - US$3/mes — "Coffee tier" (agradecimiento en README)
     - US$10/mes — "Supporter" (prioridad en issues/features)
     - Activar también **one-time payments** (mucha gente prefiere donar una vez).
5. **Esperar aprobación** de GitHub (típicamente días).
6. **Verificar**: al aprobarte, el botón *Sponsor* aparece automáticamente en `github.com/arieladasme/cs2-watch` (el `FUNDING.yml` ya está commiteado — no hay que tocar nada).
